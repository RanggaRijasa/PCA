# Private Company Assistant

This repository contains the Private Company Assistant implementation in [`company-assistant/`](company-assistant/).

The assistant is a local-first company RAG system: FastAPI backend, React TypeScript UI, SQLite metadata, ChromaDB vectors, Ollama-backed generation, permission-filtered retrieval, citations, audit logs, admin ingestion, and an evaluation harness.

Start here:

```bash
cd company-assistant
```

Then follow the full setup, run, ingestion, evaluation, backup, and handoff instructions in [`company-assistant/README.md`](company-assistant/README.md).

Codex and other coding agents should treat `company-assistant/` as the application working directory. Root-level files are only repository context and pointers.
